Introduction

Workbook 1 - Getting Started

Welcome to the first workbook for CS559!

The right way to use this workbook, and any of the following ones, is to:

  1. Clone the workbook from GitHub
  2. Serve the folder using a local server
  3. Start by looking at the index.html file in the web browser
  4. Follow the instructions

The fact that you are looking at this means you probably have done the first of those steps!

For more information on workbooks, see the Canvas (workbooks page)[https://canvas.wisc.edu/courses/167748/pages/workbooks]. For more information on using a local server, see the Canvas (tools page)[https://canvas.wisc.edu/courses/167748/pages/tools].

Note: this workbook is a little wordier and more “make sure you know how to do things” than future ones are likely to be. This workbook is to make sure you are ready to do programming. Later ones will be more about doing some programming.

In the future, we’ll be doing graphics programming. This workbook is about making sure you are ready! Future workbooks will have you do more programming, and less plain reading.

Local Server Check

You should always “run” the web pages served by a local server. See the Canvas (tools page)[https://canvas.wisc.edu/courses/167748/pages/tools] for more information. Using a local server requires both running the server and having the web browser look at the server. If you don’t use a local server (and instead you just open the file in the web browser), problems can happen.

We have no way to check if you are using a local server.

Make sure that you know how to use a local server. That’s a goal of this assignment - we will need it for future assignments. Re-read the Canvas (tools page)[https://canvas.wisc.edu/courses/167748/pages/tools], ask for help, whatever it takes.

Goals of this workbook

The learning goals of this workbook are:

  1. To make sure you can use the class mechanics (GitHub classroom links, Git, local servers, editing pages, …)
  2. To give you an idea of how the workbooks will work for the class (looking at pages, editing and adding things, turning things in)
  3. To review a little bit of basic HTML and CSS that you will need for class
  4. To look at a little bit of JavaScript, and see how we attach programs to web pages
  5. To see how we make interactions and animations with JavaScript web pages
  6. To understand event-driven- and animation-loop-driven- programming
  7. To practice writing some JavaScript and make sure you have all your tools set up

What you should do with this workbook

If you haven’t already done it, please review all the pages on the Course Website. The the Canvas (getting started page)[https://canvas.wisc.edu/courses/167748/pages/getting-started] is a good place to get started. The fact that you found this workbook and got this far means you’ve done at least some reading.

You are required to edit the README.md file (see the (workbook instructions)[https://canvas.wisc.edu/courses/167748/pages/workbooks]), so you might as well put your name, GitHub ID, and WiscID into the file now. You could commit those changes and push them back to GitHub just for practice.

The main part of this workbook is a series of pages (page 1, page 2, etc.). Each of the pages contains an instruction part and one or more “boxes”. Read through the instruction part of each of the pages first, and then move on to the boxes. Some of the boxes contain examples (including code) that you should read and make sure you understand. Most boxes also inlude exercises, and will ask you for modifications or to write code on your own and add it to the box. Each box is stored in an external HTML file in the boxes/ folder, including any imports such as JavaScript files. Those files are imported into the page HTMLs. You can find the page sources in the docs/ folder. You don’t have to understand the structure of the whole project, although you are welcome to take a deeper look at it. The page sources are generated automatically and might be hard to read. The important thing is reading and understanding the sources for the boxes, and modify them as necessary. All boxes are named according to the page and their box number on the page. For example, you can find the source file for box 3 on page 2 in boxes/page2-3.html.

You should work through the pages and their boxes in order. First, open the main page index.html in the web browser, and follow the instructions (which will usually involve opening the file in an editor, and modifying it and/or its associated JavaScript files). Remember to always use a local web server. Don’t just open the files in the browser.

And when you’re done, don’t forget to commit your work and push it back to the GitHub repo that GitHub classroom made for you!

Readings

The boxes of this workbook are required reading! That includes reading them as web pages, but also, reading them as source code.

Other readings are sprinkled across the pages of this workbook. Some things that you should make sure that you read along the way:

  • The Pages on Canvas, but particularly all the pages discussed on the getting started page.
  • Some resources on HTML and CSS (if you aren’t already familiar). You just need to know the basics. The Mozilla Learning HTML site is a good resource.
  • The tutorial on Color - is from the 2015 version of 559, but it’s still really useful.
  • If you are learning JavaScript, look at the resources on the javascript page.

For this week, there are no regular readings (book chapters). Those will start in the next week or two.

Assessment

The graders will mainly be checking to make sure you’ve followed the directions. If you’ve done everything you’re supposed to (made the required changes to files, added the required file, done the README, …) you will get the highest possible score (AB - remember that you cannot get As on workbooks).

A lot of this assignment is “self-certified” - we’ll trust you that you did what we asked you (if you turned something in, we’ll assume its correct). So for example, the graders won’t check to see what changes you made to box4-2, they will only check that you made a change to the file.

Exercises

Page 4 has required exercises on it. Each of them requires you to write your own Javascript code.

Rubric for Grading

  • Reading through the Workbook - you’ll get the points if you turn something in (at least one push to the repository) (.5 pts)
  • Exercise 4-1 (page 4, box 1) - make your own sliders (1 pts)
  • Exercise 4-2 (page 4, box 2) - animate a slider (1 pts)
  • Exercise 4-3 (page 4, box 3) - changing colors smoothly (1 pts)

Bonus Points

There are no bonus points available for this workbook.

Important: Make sure you understand everything! The ideas from this workbook will keep coming back in the future: on other workbooks, and on the exams (including JavaScript and HTML).
